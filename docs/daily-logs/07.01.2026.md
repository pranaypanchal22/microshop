## What I Learned
- A virtual environment isolates dependencies per service — pip install inside venv
  only affects that service, not the system Python or other services
- FastAPI automatically generates interactive Swagger documentation at /docs
  from your code with zero extra configuration
- uvicorn app.main:app means: find main.py inside the app/ folder,
  use the variable named app as the FastAPI instance
- __init__.py can be completely empty — its existence is what tells Python
  that the folder is a package that can be imported from
- Always run curl and project commands from Ubuntu terminal, not PowerShell —
  PowerShell wraps responses in verbose objects instead of clean JSON
