## What I Learned
- conftest.py is pytest's special config file — fixtures defined here
  are automatically available to all test files without importing them
- MockRedis replaces real Redis in tests using a plain Python dict —
  tests run faster and don't depend on an external service
- dependency_overrides replaces FastAPI dependencies (get_db, get_redis)
  with test versions — the app never knows it's being tested
- autouse=True on a fixture means it runs for every test automatically
  without being explicitly called in each test function
- Base.metadata.drop_all() after each test ensures a clean database
  state — tests never affect each other
- pytest -v shows each test name and PASSED/FAILED individually
- --cov-report=term-missing shows exactly which line numbers are
  not covered — tells you precisely what to test next
- Always use a separate test database — never run tests against
  your development or production database
